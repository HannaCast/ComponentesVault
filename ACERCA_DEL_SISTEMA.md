# Sistema de Generación de Horarios Académicos

Sistema web diseñado para que docentes universitarios puedan gestionar y generar horarios académicos de forma eficiente, organizada y personalizada según las necesidades de su institución.

---

## Descripción

Esta aplicación permite a cualquier maestro de cualquier universidad registrarse y administrar todos los elementos necesarios para construir horarios académicos: universidades, carreras, materias, grupos, profesores y aulas. Con todos estos datos configurados, el sistema es capaz de generar horarios de manera automática respetando restricciones de disponibilidad, espacios y asignaciones.

---

## Funcionalidades principales

- **Autenticación de usuarios** — Registro e inicio de sesión con datos personales del docente.
- **Gestión de universidades** — Agrega, edita y selecciona la universidad activa sobre la cual se trabajará.
- **Gestión de carreras** — Administra las carreras de la universidad, sus periodos y materias asignadas por semestre.
- **Gestión de materias** — Configura materias con su color, horas por semana y profesores que pueden impartirla.
- **Gestión de grupos** — Define grupos por carrera, periodo, turno y letra.
- **Gestión de profesores** — Registra profesores con sus intervalos de disponibilidad y materias que imparten.
- **Gestión de aulas** — Administra aulas por tipo, edificio, piso y restricciones por carrera.
- **Generación de horarios** — Genera horarios académicos automáticamente en base a todos los datos configurados.
- **Ajustes de cuenta** — El usuario puede actualizar sus datos personales y contraseña.

---

## Pantallas del sistema

| Pantalla | Descripción |
|---|---|
| Login / Registro | Acceso y creación de cuenta |
| Universidades | Selección y administración de instituciones |
| Carreras | Gestión de carreras y sus periodos |
| Materias | Administración de materias por carrera |
| Grupos | Configuración de grupos por turno y periodo |
| Profesores | Gestión de docentes y disponibilidad |
| Aulas | Administración de espacios físicos |
| Generar Horario | Generación automática de horarios |
| Ajustes | Configuración de la cuenta del usuario |

## Tecnologías utilizadas

- **Frontend:** React
- **Backend:** Django REST Framework (API)
- **Base de datos:** MySQL
