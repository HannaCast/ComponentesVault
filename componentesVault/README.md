# Infinity Vault Components

Base de componentes React para una app tipo Drive/almacenamiento en la nube.

## Uso

Los componentes principales se exportan desde:

```js
import { VaultButton, FileCard, VaultSidebar } from '@components';
```

## Componentes incluidos

- Branding: `VaultBrand`, `VaultIconBadge`.
- Acciones y formularios: `VaultButton`, `VaultTextField`, `VaultSearchBar`, `VaultNewItemMenu`.
- Navegacion: `VaultSidebar`, `VaultTopbar`, `ViewToggle`.
- Contenido: `MetricCard`, `FileCard`, `WorkspaceCard`, `PlanCard`, `TrashTable`, `StorageUsage`, `VaultFeatureCard`.

Los estilos dependen de los tokens CSS definidos en `src/index.css`.
